# PCB制作 现代


# 🌀 电风扇控制板（PCB）主要模块说明

| 模块名称 | 功能说明 | 关键元件 / 芯片 | 与其他模块连接 | 设计要点 / 注意事项 | 常见故障与排查 |
|-----------|-----------|------------------|----------------|----------------------|----------------|
| **电源模块 (Power Supply)** | 将市电 AC 220V 转为低压 DC（3.3V / 5V / 12V）供全板使用 | 保险丝、压敏电阻(MOV)、整流桥(DB107)、滤波电容、电源芯片(AMS1117 / LM7805) | 向 MCU、驱动、显示模块供电 | 注意高低压隔离、滤波、电容耐压与温升控制 | 不上电 / 电容鼓包 / 重启：检查整流桥、电容、稳压芯片 |
| **主控模块 (MCU)** | 控制风扇全部逻辑，如按键响应、风速调节、定时、显示等 | STC、Holtek、STM8、PIC、Nuvoton MCU、晶振、电容 | 与输入、驱动、显示模块相连 | MCU 引脚去耦、地平面完整、软件去抖、防 EMI | 按键无响应 / 逻辑混乱：检查电源稳定与复位电路 |
| **按键输入模块 (Buttons)** | 接收用户操作（开/关、风速、定时、摇头） | 轻触按键、电阻阵列 | 连接 MCU GPIO 输入脚 | 添加 RC 去抖，矩阵布线短且避开高压区 | 按键失灵或乱跳：检查电阻焊点、键帽接触 |
| **红外接收模块 (IR Receiver)** | 接收遥控器信号 | HS0038 / VS1838 模块 | 连接 MCU 外部中断脚 | 远离电源噪声、信号线加滤波电容 | 遥控距离短 / 无响应：检查电源滤波或红外头信号输出 |
| **电机驱动模块 (Motor Driver)** | 控制电机启停、风速、方向 | AC：继电器 / TRIAC 可控硅 / 光耦；DC：MOSFET / 驱动芯片 | 连接电机与 MCU 控制端 | 高低压隔离、续流二极管、功率裕量、散热设计 | 电机不转 / 抖动：检测驱动波形、继电器吸合、MOS 管温度 |
| **摇头驱动模块 (Oscillation Motor)** | 控制摇头电机或电磁机构 | 继电器 / 三极管 / MOS 管、续流二极管 | 连接 MCU 与摆头电机 | 电流裕量足够、反向电压保护 | 摆头不动 / 无力：检查继电器供电与线圈电压 |
| **显示与指示模块 (Display / LED)** | 显示风速、定时、状态、声音提示 | LED 指示灯、数码管、蜂鸣器、驱动芯片（ULN2003） | 连接 MCU 输出端 | 共阴/共阳匹配、限流电阻、电流分配 | LED 不亮 / 错亮：检查限流电阻与驱动管脚电压 |
| **保护与滤波模块 (Protection / EMI)** | 抑制浪涌与电磁干扰，保护电路安全 | 压敏电阻(MOV)、共模电感、安规电容(Y 电容) | 串联于电源输入端 | 接地良好、高低压安全距离 | 开机干扰大 / 板子发热：检查滤波元件或接地 |
| **连接接口模块 (Connectors)** | 提供外部连接（电源线、电机、按键板、显示板） | 接线端子、排针、排线插座 | 连接整机外部组件 | 插针方向、极性标识清晰，防呆设计 | 插错 / 虚焊：检查接头焊点、极性标识 |

---

> 💡 **总结：**
> 电风扇控制板核心逻辑是「电源稳定 → MCU 控制 → 电机驱动 → 用户交互（按键/显示）」，  
> 整体以低成本、高稳定性为优先设计目标。
